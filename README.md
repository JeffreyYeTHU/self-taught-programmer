### Java

缺一本可以推荐的入门书，Bruce Eckel前后写的`Prentice.Hall.Core.Java.Volume.I.Fundamentals.11th.Edition`和`On Java 8`都太厚了，适合当手册查阅，不适合通读。

缺入门的书，看到的入门书都太厚，不太适用。
